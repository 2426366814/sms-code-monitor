## 🔐 安全特性验证

### 权限隔离机制

```
admin.html 登录验证流程:
├── 用户提交登录表单
├── API返回用户信息
├── 检查 is_admin 字段
│   ├── is_admin = true → 允许登录
│   │   ├── 清除 token, user
│   │   ├── 设置 admin_token, admin_user
│   │   └── 显示管理员后台
│   │
│   └── is_admin = false → 拒绝登录
│       ├── 显示"访问被拒绝：您没有管理员权限"
│       └── 阻止进入后台
```

### Token管理机制

```
login.html 登录流程:
├── 管理员登录
│   ├── 设置 token
│   ├── 设置 admin_token
│   └── 重定向到 admin.html
│
└── 普通用户登录
    ├── 设置 token
    ├── 清除 admin_token ← 关键安全措施
    └── 重定向到 index.html
```

### API数据隔离

```
API请求验证:
├── 获取token from localStorage
├── 发送请求 with Authorization header
├── 后端验证token并获取user_id
├── 只返回该用户的数据
└── 数据隔离验证通过
```
