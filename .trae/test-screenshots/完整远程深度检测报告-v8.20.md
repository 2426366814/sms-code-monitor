## 🔐 安全特性验证

### 权限隔离机制

```
admin.html 访问验证流程:
├── 检查 localStorage 中的 admin_token
├── 如果 admin_token 不存在
│   └── 重定向到 login.html
└── 如果 admin_token 存在
    └── 显示管理员后台
```

### Token管理机制

```
login.html 登录流程:
├── 管理员登录
│   ├── 设置 token
│   ├── 设置 admin_token
│   └── 重定向到 admin.html
│
└── 普通用户登录
    ├── 设置 token
    ├── 清除 admin_token ← 关键安全措施
    └── 重定向到 index.html
```

### API认证验证

```
API请求验证:
├── 获取token from localStorage
├── 发送请求 with Authorization header
├── 后端验证token
│   ├── token有效 → 返回数据
│   └── token无效/缺失 → 返回401
└── 数据隔离验证通过
```
