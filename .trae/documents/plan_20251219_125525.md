# 修复API密钥生成失败和API链接优化

## 问题分析
1. **API密钥生成失败**：系统在尝试生成API密钥时失败，错误信息是"SQLSTATE[42S02]: Base table or view not found: 1146 Table 'jm.api_keys' doesn't exist"。这说明数据库中没有api_keys表，导致API密钥生成失败。
2. **API链接缺少密钥**：用户希望API链接应当带上密钥，方便外部访问。

## 解决方案
1. **确保api_keys表被成功创建**：修改DatabaseInit.php文件，确保api_keys表被成功创建。
2. **优化API链接**：修改API链接生成逻辑，使其包含API密钥，方便外部访问。

## 修复步骤
1. 修改`backend/utils/DatabaseInit.php`文件：增强数据库初始化逻辑，确保api_keys表被成功创建
2. 修改`frontend/admin/system.html`文件：优化API链接生成逻辑，使其包含API密钥
3. 测试修改后的效果，确保API密钥生成成功，API链接包含密钥
4. 重新打包项目

## 预期结果
修复后，系统将能够成功生成API密钥，并且API链接将包含API密钥，方便用户复制和外部访问。
