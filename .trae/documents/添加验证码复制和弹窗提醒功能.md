## 功能需求

1. **一键复制验证码功能**
   - 在监控列表和历史记录中的验证码显示区域添加复制按钮
   - 点击后自动复制验证码到剪贴板
   - 显示复制成功的提示

2. **最新验证码弹出提醒功能**
   - 当系统获取到新的验证码时，自动弹出提醒窗口
   - 弹窗显示手机号、验证码、接收时间
   - 弹窗支持一键复制验证码
   - 弹窗自动消失或手动关闭

## 实现方案

### 1. 一键复制功能
- 修改 `.code-display` 样式，添加复制按钮
- 使用 Clipboard API 实现复制功能
- 添加复制成功的 Toast 提示
- 在监控列表表格和历史记录卡片中都添加复制功能

### 2. 验证码弹窗提醒
- 创建新的弹窗组件 `notification-modal`
- 在 `loadMonitors()` 和自动刷新时检测新验证码
- 使用 `localStorage` 记录已显示的验证码ID，避免重复提醒
- 弹窗包含：手机号、验证码（大字体）、时间、复制按钮、关闭按钮
- 弹窗5秒后自动消失，或点击关闭按钮立即关闭
- 支持桌面通知（如果用户开启）

### 3. 需要修改的文件
- `index.html` - 用户面板
